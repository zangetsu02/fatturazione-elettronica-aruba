# Installation

## Prerequisites

- **Node.js** >= 18.0.0
- An account on [Aruba Electronic Invoicing](https://fatturazioneelettronica.aruba.it)

## Available Packages

The SDK is composed of modular packages. Install only what you need:

| Package | Description |
|---------|-------------|
| `@fatturazione-elettronica-aruba/core` | HTTP client, authentication, base types |
| `@fatturazione-elettronica-aruba/invoices` | Invoice management (send/receive) |
| `@fatturazione-elettronica-aruba/notifications` | SDI notifications |
| `@fatturazione-elettronica-aruba/communications` | Financial communications |
| `@fatturazione-elettronica-aruba/utils` | Utilities (Base64, dates, constants) |

## Basic Installation

For most use cases:

::code-group
```bash [pnpm]
pnpm add @fatturazione-elettronica-aruba/core @fatturazione-elettronica-aruba/invoices
```

```bash [npm]
npm install @fatturazione-elettronica-aruba/core @fatturazione-elettronica-aruba/invoices
```

```bash [yarn]
yarn add @fatturazione-elettronica-aruba/core @fatturazione-elettronica-aruba/invoices
```
::

## Full Installation

For all features:

::code-group
```bash [pnpm]
pnpm add @fatturazione-elettronica-aruba/core \
  @fatturazione-elettronica-aruba/invoices \
  @fatturazione-elettronica-aruba/notifications \
  @fatturazione-elettronica-aruba/communications \
  @fatturazione-elettronica-aruba/utils
```

```bash [npm]
npm install @fatturazione-elettronica-aruba/core \
  @fatturazione-elettronica-aruba/invoices \
  @fatturazione-elettronica-aruba/notifications \
  @fatturazione-elettronica-aruba/communications \
  @fatturazione-elettronica-aruba/utils
```
::

## Verify Installation

```typescript
import { ArubaClient } from '@fatturazione-elettronica-aruba/core';

const client = new ArubaClient({
  environment: 'demo',
});

console.log('SDK installed successfully!');
```
