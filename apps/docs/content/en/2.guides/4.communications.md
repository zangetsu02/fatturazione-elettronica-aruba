# Financial Communications

Send communications to the Italian Revenue Agency.

## Setup

```typescript
import { CommunicationsClient } from '@fatturazione-elettronica-aruba/communications';

const communications = new CommunicationsClient(client.http);
```

## Communication Types

| Code | Description |
|------|-------------|
| `LI` | Periodic VAT settlements |
| `DTE` | Issued invoice data |
| `DTR` | Received invoice data |
| `ANN` | Cancellation |

## Send Communication

```typescript
import { encodeBase64 } from '@fatturazione-elettronica-aruba/utils';

const result = await communications.createTransmission({
  comunicationType: 'LI',
  dataFile: encodeBase64(xmlContent),
});

console.log('Request ID:', result.requestId);
```

## Check Status

```typescript
const info = await communications.getTransmissionInfo(result.requestId);

console.log('Status:', info.result);
console.log('Result:', info.elaboratedResult);
```

## Status Codes

| Code | Description |
|------|-------------|
| `SF01` | File received |
| `SF02` | File processing |
| `SF03` | File processed |
| `ES01` | File accepted |
| `ES02` | File accepted with warnings |
| `ES03` | File rejected |
