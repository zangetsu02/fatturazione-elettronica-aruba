# Core

Main package with HTTP client, authentication and base types.

## ArubaClient

```typescript
import { ArubaClient } from '@fatturazione-elettronica-aruba/core';

const client = new ArubaClient(options);
```

### Options

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `environment` | `'demo' \| 'production'` | - | API environment |
| `timeout` | `number` | `30000` | Request timeout (ms) |
| `maxRetries` | `number` | `3` | Retries for 429 errors |
| `tokenStorage` | `TokenStorage` | - | Custom token storage |

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `auth` | `AuthClient` | Authentication client |
| `http` | `HttpClient` | HTTP client |

## AuthClient

### signin(username, password)

```typescript
const result = await client.auth.signin('user', 'pass');
```

### refresh(refreshToken)

```typescript
const newToken = await client.auth.refresh(token);
```

### getUserInfo()

```typescript
const info = await client.auth.getUserInfo();
```

### getMulticedenti(params)

```typescript
const result = await client.auth.getMulticedenti({ page: 0, size: 20 });
```
