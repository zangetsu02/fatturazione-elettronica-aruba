# Invoices

Electronic invoice management.

## InvoicesClient

```typescript
import { InvoicesClient } from '@fatturazione-elettronica-aruba/invoices';

const invoices = new InvoicesClient(httpClient);
```

## Methods

### upload(request)

Send unsigned invoice.

### uploadSigned(request)

Send already signed invoice.

### findSent(params)

Search sent invoices.

### getSentDetail(params)

Get sent invoice detail.

### findReceived(params)

Search received invoices.

### getReceivedDetail(params)

Get received invoice detail.

### sendEsitoCommittente(request)

Send acceptance/rejection outcome.

### getSentZip(params)

Download invoice ZIP.

### getSentPdd(params)

Download PDD (Distribution Package).

### downloadSent(ids, types)

Bulk download.

### getDownload(jobId)

Get bulk download result.
