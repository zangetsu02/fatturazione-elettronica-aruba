# Utils Package

The `@fatturazione-elettronica-aruba/utils` package provides helper functions and constants for common operations when working with the Aruba API.

## Installation

```bash
pnpm add @fatturazione-elettronica-aruba/utils
```

## Base64 Functions

### encodeBase64

Encodes a string or binary data to Base64. Used when uploading invoices to the API.

```typescript
function encodeBase64(data: string | Uint8Array): string
```

**Parameters:**
| Name | Type | Description |
|------|------|-------------|
| `data` | `string \| Uint8Array` | The data to encode |

**Returns:** Base64 encoded string

**Example:**
```typescript
import { encodeBase64 } from '@fatturazione-elettronica-aruba/utils';
import fs from 'node:fs';

// Encode string
const xmlString = '<FatturaElettronica>...</FatturaElettronica>';
const encoded = encodeBase64(xmlString);

// Encode file buffer
const fileBuffer = fs.readFileSync('invoice.xml');
const encodedFile = encodeBase64(new Uint8Array(fileBuffer));
```

### decodeBase64

Decodes a Base64 string to binary data (Uint8Array). Used when downloading invoices from the API.

```typescript
function decodeBase64(base64: string): Uint8Array
```

**Parameters:**
| Name | Type | Description |
|------|------|-------------|
| `base64` | `string` | The Base64 string to decode |

**Returns:** Decoded data as `Uint8Array`

**Example:**
```typescript
import { decodeBase64 } from '@fatturazione-elettronica-aruba/utils';
import fs from 'node:fs';

// Decode and save to file
const base64Data = response.file;
const binaryData = decodeBase64(base64Data);
fs.writeFileSync('downloaded-invoice.xml', binaryData);
```

### decodeBase64ToString

Decodes a Base64 string directly to a UTF-8 string.

```typescript
function decodeBase64ToString(base64: string): string
```

**Parameters:**
| Name | Type | Description |
|------|------|-------------|
| `base64` | `string` | The Base64 string to decode |

**Returns:** Decoded UTF-8 string

**Example:**
```typescript
import { decodeBase64ToString } from '@fatturazione-elettronica-aruba/utils';

const xmlContent = decodeBase64ToString(response.file);
console.log(xmlContent); // <?xml version="1.0"?>...
```

## Date Functions

### formatDateISO

Formats a Date object to ISO 8601 format.

```typescript
function formatDateISO(date: Date): string
```

**Returns:** `2024-01-15T10:30:00.000Z`

### formatDateOnly

Formats a Date object to date-only format (YYYY-MM-DD).

```typescript
function formatDateOnly(date: Date): string
```

**Returns:** `2024-01-15`

**Example:**
```typescript
import { formatDateOnly } from '@fatturazione-elettronica-aruba/utils';

const date = new Date();
const searchDate = formatDateOnly(date);

await invoices.findSent({
  creationStartDate: searchDate,
  creationEndDate: searchDate,
});
```

### formatDateAruba

Formats a Date to Aruba's specific datetime format.

```typescript
function formatDateAruba(date: Date): string
```

**Returns:** `2024-01-15 10:30:00`

### parseArubaDate

Parses a date string in Aruba format to a Date object.

```typescript
function parseArubaDate(dateString: string): Date
```

**Parameters:**
| Name | Type | Description |
|------|------|-------------|
| `dateString` | `string` | Date in format `YYYY-MM-DD HH:mm:ss` |

**Returns:** JavaScript `Date` object

### getDateRange

Generates a date range from N days ago until today.

```typescript
function getDateRange(days: number): { startDate: string; endDate: string }
```

**Parameters:**
| Name | Type | Description |
|------|------|-------------|
| `days` | `number` | Number of days to go back |

**Returns:** Object with `startDate` and `endDate` in ISO format

**Example:**
```typescript
import { getDateRange } from '@fatturazione-elettronica-aruba/utils';

// Get invoices from the last 30 days
const { startDate, endDate } = getDateRange(30);

await invoices.findSent({
  creationStartDate: startDate,
  creationEndDate: endDate,
});
```
